## Overview
This project analyzes Netflix dataset to explore trends in genres, ratings, and durations.

## Steps to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Download the dataset from Kaggle:
   https://www.kaggle.com/datasets/shivamb/netflix-shows

3. Place `netflix_titles.csv` in the same folder as `app.py`

4. Run the Streamlit dashboard:
   ```bash
   streamlit run app.py
   ```

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Streamlit
